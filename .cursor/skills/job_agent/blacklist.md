# Job Blacklist

This file contains jobs you've reviewed and don't want to see again in future searches.

## Three-Level Blacklist System

### 1. Blacklisted Companies
Block **ALL** jobs from these companies (company-wide ban)

### 2. Blacklisted Job Titles
Block **all postings** with this specific job title at this company, regardless of URL
Use when you never want to see this type of role at this company

### 3. Blacklisted Job Postings (URLs)
Block **only specific posting URLs** (e.g., expired/broken links)
Same job title with a new URL will still appear

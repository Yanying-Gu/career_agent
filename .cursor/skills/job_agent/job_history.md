# Job Search History

This file tracks jobs you're interested in and want to save for later review or application.

**Manual tracking:** Add jobs here that you like from search results.

## Format

**Format:** **Company** - Job Title [YYYY-MM-DD] {status} (link)  
*Note: Only company name and job title are required. Date, status, and link are optional.*

**For multiple links:**  
Use labeled links: (LinkedIn: url) (Company: url) (Other: url)

**Status options:** interested | applied | interviewing | offer | rejected | expired | archived
